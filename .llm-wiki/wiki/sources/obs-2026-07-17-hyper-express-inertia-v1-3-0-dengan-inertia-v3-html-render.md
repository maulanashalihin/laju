# ⭐ Observation: hyper-express-inertia v1.3.0 dengan Inertia v3 HTML render
Updated hyper-express-inertia package (v1.3.0) dari Inertia v2-style HTML render ke native v3 format. Perubahan: <div id="app"> tanpa data-page, page JSON pindah ke <script data-page="app" type="application/json">, stylesheet di antara JSON script dan JS script. Frontend @inertiajs/svelte otomatis mendukung format baru ini. Published ke npm.
