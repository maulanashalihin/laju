Powerful Features

Everything you need to build modern web applications

Lightning Fast

Built on HyperExpress for exceptional performance. 2.08x faster than pure Node.js with 58% lower latency.

Modern Stack

Svelte 5 + Inertia.js for seamless client-server communication. TypeScript for type safety.

Database Performance

BetterSQLite3 with WAL mode. Up to 19.9x faster writes and 385% faster reads.

Built-in Auth

Google OAuth integration, session management, and email verification out of the box.

Modern Tooling

Vite for fast builds, TailwindCSS for styling, and Squirrelly for server-side rendering.

AI-Ready

100% no-code workflow with AI assistants. Build complete apps using Cursor, Windsurf, or Trae.ai.
