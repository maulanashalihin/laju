Get Started in Seconds

// app/controllers/HomeController.ts
import { Response, Request } from "../../type";
import { view } from "../services/View";

class Controller {
    public async index(request: Request, response: Response) {
        return response.type("html").send(view("index.html"));
    }
}

export default new Controller();

// routes/web.ts
import AuthController from "../app/controllers/AuthController"; 
import HyperExpress from 'hyper-express';

const Route = new HyperExpress.Router();
 
Route.get("/", HomeController.index);

Just a few lines of code to get your server up and running. No complex configurations, no boilerplate.

FAQ

Apa itu Laju?
Laju adalah framework TypeScript full‑stack yang menggabungkan HyperExpress (server ultra‑cepat), Svelte 5, Inertia.js, TailwindCSS, Vite, dan template engine Squirrelly untuk pengembangan cepat dengan DX yang nyaman.
Bagaimana cara memulai dengan cepat?
Jalankan npx create-laju-app my-app, lalu npm run dev. Tidak perlu konfigurasi rumit.
Apa yang membuat Laju sangat cepat?
Kombinasi HyperExpress, Vite (hot module + build cepat), dan Squirrelly (template engine ringan & kencang) dengan arsitektur MVC yang sederhana.
Apakah ada autentikasi bawaan?
Ya. Tersedia login, register, forgot/reset password, dan integrasi Google OAuth.
Dukungan database dan migrasi?
Menggunakan Knex.js untuk query builder dan migrasi, dengan BetterSQLite3 secara default. Perintah: npx knex migrate:latest.
Bagaimana cara deploy?
npm run build. Untuk produksi, gunakan PM2 runtime dengan konfigurasi di README.
